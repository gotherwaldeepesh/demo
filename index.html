Description


Library management system is a multi-user and single admin, web application. Users need to register first to log in and use the app but librarian registration is hard-coded. In the app, a user can issue a maximum of 5 eBooks for seven days, download any eBook for a price, and write comments for issued eBooks. An admin can add, update, or delete a section or an eBook, accept or reject issue requests.


Technologies used


  • Flask: for basic backend Implementation.
  • Flask_sqlalchemy: for creating Database.
  • Some inbuilt libraries like jinja2, render_template, redirect, and others displaying HTML content.
  • flash: to show an alert.
  • Flask_Login: for implementing the login functionality.
  • werkzeug.security: for hashing and checking the password.
  • Flask-Restful: to create Rest Api.
